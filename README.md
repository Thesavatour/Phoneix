 

##  Fin - Funder a Trading App

First, clone this repo then run this command:

```bash
npm install
# or
yarn 
```

 Then, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
